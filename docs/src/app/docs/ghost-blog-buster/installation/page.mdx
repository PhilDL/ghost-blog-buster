# Quickstart

This tool can be installed globally or used directly with `npx`.

<Steps>

### Install globally

```bash title="terminal"
npm install -g @ts-ghost/ghost-blog-buster
```

### Use in your terminal

Then, in a new terminal session, run the CLI

```bash title="terminal"
ghost-blog-buster
```

This will take you to the interactive version of the tool.

![Screen Shot](https://user-images.githubusercontent.com/4941205/221599018-60f66258-9cfa-459a-928d-907c2df01f0e.gif)

</Steps>

## Alternatively you can use npx to run the tool without installing it.

```bash title="terminal"
npx @ts-ghost/ghost-blog-buster
```
